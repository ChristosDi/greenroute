# GreenRoute

GreenRoute is a sustainable urban transport web application that helps users track the carbon emissions of their journeys and make lower-impact travel choices.

## Live Demo

[View GreenRoute on Render](https://greenroute-quwy.onrender.com)

## What It Does

- Tracks journeys and estimates related carbon emissions.
- Supports transport and vehicle-based emissions calculations.
- Uses elevation data to improve journey gradient estimates.
- Provides account-based access with MongoDB-backed data.

## Tech Stack

- Node.js
- Express
- MongoDB
- EJS
- Render

## Project Structure

The application source code is inside the `greenroute/` directory. Deployment is configured with `render.yaml` for Render Blueprint deployment.
